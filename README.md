\# 🚌 Cloud-Based Bus Pass System



\## 📌 Project Overview



The Cloud-Based Bus Pass System is an online bus pass booking application developed to provide a secure and reliable digital ticketing experience.



The system allows users to register, login, book bus passes, generate digital tickets, and verify tickets using QR codes.



This project focuses on improving scalability, reliability, security, and reducing problems such as ticket loss, theft, and incorrect pricing.



---



\## 🚀 Features



✅ User Registration and Login
